# Social Politician Landing Page

Landing page for **socialpolitician.com** that links to all app pages at **app.socialpolitician.com**.

## Breakdance Builder – Quick Start

### Option A: All-in-one (easiest)
1. Copy the entire contents of `home.html`
2. Add a **Breakdance HTML** element to your page
3. Paste the HTML
4. Done – styles are embedded

### Option B: Separate CSS
1. Upload `style.css` to your theme (e.g. `/wp-content/themes/your-theme/socialpolitician.css`)
2. Enqueue it in WordPress, or add via Breakdance > Custom CSS
3. Copy `home.html` but remove the `<style>...</style>` block
4. Paste into a Breakdance HTML element

## Customization

- All classes use `sp-` prefix to avoid WordPress theme conflicts
- Colors: `--sp-navy`, `--sp-cream`, `--sp-red` in the CSS
- Responsive: uses `clamp()` for fluid typography and spacing
